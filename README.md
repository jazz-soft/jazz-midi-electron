## MIDI for Electron

This module is for [**Electron**](https://electronjs.org) projects only!  
Please use [**jazz-midi**](https://www.npmjs.com/package/jazz-midi) for command-line **Node.js** applications.

### Install
`npm install jazz-midi-electron --save`
